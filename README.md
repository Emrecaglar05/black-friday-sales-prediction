## 🌑 Proje Hakkında

> **Black Friday Sales Predictor**, büyük ölçekli bir perakende veri setini kullanarak müşteri satın alma tutarlarını tahmin eden uçtan uca bir makine öğrenmesi projesidir. Gerçek dünya verisiyle kapsamlı EDA, ileri düzey özellik mühendisliği ve Optuna ile Bayesian hiperparametre optimizasyonu içerir.

Bu proje; veri analizi, görselleştirme ve tahmin modelleme adımlarını retro-tech cyberpunk görsel teması altında bir araya getirir. 12 farklı model denenmiş, en iyi sonuç optimize edilmiş **LightGBM** ile elde edilmiştir.

```
📂 Proje Kapsamı
├── 🔍 Keşifsel Veri Analizi (EDA)      → Demografik & Satın Alma Örüntüleri
├── 🛠️  Özellik Mühendisliği (FE)        → 5 Farklı FE Modülü, 40+ Özellik
├── 🤖  12 Model Karşılaştırması         → Baseline → FE → Tuned Pipeline
├── ⚙️  Optuna Bayesian Optimizasyon     → LightGBM / XGBoost / CatBoost
└── 📤  Final Tahmin & Submission        → Bimodal Müşteri Segmentasyonu
```

## 🗂️ Veri Seti

Proje, Kaggle'ın popüler **Black Friday** veri setini kullanmaktadır.

| Değişken | Tür | Açıklama |
|---|---|---|
| `User_ID` | Kategorik | Benzersiz müşteri kimliği |
| `Product_ID` | Kategorik | Benzersiz ürün kimliği |
| `Gender` | Kategorik | M / F |
| `Age` | Kategorik | 7 yaş grubu (0-17 → 55+) |
| `Occupation` | Numerik | Meslek kodu (0–20) |
| `City_Category` | Kategorik | A / B / C şehir tipi |
| `Stay_In_Current_City_Years` | Ordinal | 0 / 1 / 2 / 3 / 4+ yıl |
| `Marital_Status` | Binary | 0: Bekar / 1: Evli |
| `Product_Category_1,2,3` | Numerik | Hiyerarşik kategori yapısı |
| **`Purchase`** ⭐ | **Hedef** | **Satın alma tutarı (USD)** |

> 📌 **550,000+ satır** · **11 özellik** · **1 hedef değişken**

---

## 🛠️ Özellik Mühendisliği Pipeline

```python
run_fe_preprocessing(train_df, test_df)
# ├── FE-1: Category Combination Features   → cat_count, cat1_cat2_combo
# ├── FE-2: User Aggregation Features       → user_mean, user_total, user_txn_count
# ├── FE-3: Product Aggregation Features    → prod_mean, prod_std, prod_popularity
# ├── FE-4: Interaction Features            → age×city, gender×cat1, occupation×city
# └── FE-5: Encoding + Target Encoding      → K-Fold Target Encoding + log1p(Purchase)
```

## 🚀 Kurulum

### Gereksinimler

```bash
Python >= 3.10
```

### 1. Repoyu Klonla

```bash
git clone https://github.com/kullanici-adi/blackfriday-sales-predictor.git
cd blackfriday-sales-predictor
```

### 2. Sanal Ortam Oluştur (Opsiyonel ama Önerilir)

```bash
python -m venv venv
source venv/bin/activate        # Linux / macOS
venv\Scripts\activate           # Windows
```

### 3. Bağımlılıkları Yükle

```bash
pip install -r requirements.txt
```

### 4. Veriyi İndir

Veri setini [Kaggle Black Friday](https://www.kaggle.com/datasets/mehdidag/black-friday) sayfasından indirip `data/` klasörüne koy:

```
data/
├── train.csv
└── test.csv
```

### 5. Notebook'u Çalıştır

```bash
jupyter notebook blackfriday.ipynb
```
